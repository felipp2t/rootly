## API (`apps/api`)
Fastify 5, Drizzle ORM + PostgreSQL, Zod 4, JWT (`jose`) + Argon2 (`argon2` pkg), Vitest (+ `@testcontainers/postgresql` for e2e), Biome. ESM (`"type": "module"`), `tsx` for dev/scripts. OpenAPI via `@fastify/swagger` + `fastify-type-provider-zod`, exported by `src/openapi.ts`. Files: storage via `minio` client. Realtime: `@fastify/websocket`.

## Web (`apps/web`)
React + Vite 7, TanStack Router/Query/Form, shadcn/ui + Tailwind CSS 4, Zod. HTTP client fully generated by **orval** (react-query mode, axios-based, custom mutator `fetchWithAuth`) from the API's `openapi.json` — see `mem:web/core`.

Note: root `CLAUDE.md` says "React 18" but `pnpm-workspace.yaml` catalog actually pins `react: ^19.2.0` / `@types/react: ^19.2.7` — trust the catalog over the doc if versions matter.

## Package manager
pnpm 11 workspaces (`packages: apps/**`), shared dep versions pinned via `catalog:` in `pnpm-workspace.yaml` (biome, react, react-dom, typescript, vite, @types/node, @vitejs/plugin-react).
